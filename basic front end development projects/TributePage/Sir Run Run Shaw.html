Sir. Run Run Shao

The man who build up the social media empire of HK

Sir.Run Run Shao, who build up the movie empire of HK when he is 40 years old. When he found the trend was changing, he changed the bussiness mode immediately, close the movie business and start social media business.

Here's a time line of Sir. Run Run Shao's life:

  • Born in Shanghai, November 1907.
  • Start up Shaw Brothers film studio in 1958.
  • Was canonized as the knight bachelor by Queen Elizabeth II in 1977.
  • Donated more than ten billion in social welfare, especially to education and science and technology during his lifetime. Many colleges and universities have Chinese Run Run Shaw named "Yifu building".
  • n the late years, Run Run Shaw allocated funds to establish the "Run Run Shaw prize", which was awarded the "Oriental Nobel prize", in recognition of the outstanding contributions in education and scientific research.
  • Dies in home quietly at the age of 107 in 2014
